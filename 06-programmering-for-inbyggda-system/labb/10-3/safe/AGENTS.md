# Codex Local Rules

- Codex får inte ändra, skapa, ta bort eller skriva om filer i detta projekt utan att användaren uttryckligen säger att en sådan ändring ska göras.
- Codex får inte köra kommandon som förändrar filer, byggartefakter eller projektets tillstånd utan användarens explicita godkännande.
- Om användarens begäran är oklar ska Codex stanna vid analys, läsning och förslag tills användaren tydligt ber om en ändring.
