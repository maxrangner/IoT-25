# Inlämning 05: Algoritmer, Datastrukturer och Design Patterns

## Features
* `Circular queue` - Ringbuffer för temporär förvaring av nya Events
* `Linked list` - Logg för att spara data långsiktigt
* `Observer` - Utvärderar alarm och flyttar in Events till logg.

Kompileras med:
`gcc -Iinclude src/*.c -o eventlogger`
